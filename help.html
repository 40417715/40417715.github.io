About

Jotes is an innovative tool, allowing you to save, edit and delete notes in real time. By logging into Jotes you can see the collection of your "jotes".

Using the notes

Once logged in, you will be allowed access to the "jotes" page one. You will be able to view previous notes that you have saved. If you want to add a new note, click the + button and enter the details. You can then edit, view and delete the note using the button placed at the end column. Then click the save button.
